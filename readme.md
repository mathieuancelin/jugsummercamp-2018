## Run

```sh
rm -rf RUNNING_PID
docker-compose build
docker-compose up
yarn install
yarn build-front-dev
yarn start
java -Dapp.privateapps.port=8080 -Dapp.domain=jsc.fr -Dapp.importFrom=./config/otoroshi.json -Dapp.webhooks.size=20 -jar $OTO_BIN/otoroshi.jar
```
